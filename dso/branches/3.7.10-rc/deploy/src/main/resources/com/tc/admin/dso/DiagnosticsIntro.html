Terracotta Developer Console: Diagnostics

Contains lock analysis, cluster-wide thread dump, and statistics gathering tools to diagnose problems and recognize and interpret trends.

Lock profiler
Analyze lock-use patterns and statistics to help diagnose performance issues and failures. Locate deadlocks, expose code with low locality of reference, pinpoint areas of high lock contention.
Cluster thread dumps
Take a thread dump for individual clients, server instances, or the entire cluster. Store, search, and export thread dumps.
Cluster statistics recorder
Configure and manage the recording of statistics for your entire cluster. Obtain detailed information about every node in a running Terracotta cluster and correlate the results. Start, stop, view, and export recording sessions. View recorded information in the Snapshot Visualization Tool.
