Terracotta Server Nonstop Sample
==========

Overview:
=========
This example will demonstrate the non-stop and the rejoin features of Terracotta Server

Requirements:
===============
Be sure to have the JAVA_HOME defined.
For Windows users, you can use the following command:
set JAVA_HOME="path/to/java/home"
For UNIX/Linux users, you can use the following command:
export JAVA_HOME="path/to/java/home"


Running the Example:
====================
1. Windows users
    Launch the server with the start-sample-server.bat
    Run run.bat
    Validate the first step
    Stop the server with the stop-sample-server.bat
    Validate the second step
    Launch the server with the start-sample-server.bat and wait until launch is complete
    Validate the third step
    Stop the server with the stop-sample-server.bat

2. UNIX/Linux users
    Launch the server with the start-sample-server.sh
    Run run.sh
    Validate the first step
    Stop the server with the stop-sample-server.sh
    Validate the second step
    Launch the server with the start-sample-server.sh and wait until launch is complete
    Validate the third step
    Stop the server with the stop-sample-server.sh
