Terracotta Developer Console: Platform

Contains detailed information about the underlying Distributed Shared Objects platform, used as the basis for the Express Ehcache, Quartz, and Sessions clustering product suites. Browse the Clustered Heap of shared objects and access Diagnostics that can be critical to understanding the operation of your cluster.

Clustered heap
Contains tools for monitoring shared objects in the clustered heap. The clustered heap is the memory shared with Terracotta across JVMs. Shared object graphs are found in the clustered heap.
Diagnostics
Contains lock analysis, cluster-wide thread dump, and statistics gathering tools to diagnose problems and recognize and interpret trends.
